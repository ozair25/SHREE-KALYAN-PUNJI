# Shri Kalyan Punji

A premium, elegant, and secure financial advisory and capital solutions platform designed for clients across India and globally. Established in 2012, Shri Kalyan Punji delivers reliable capital flow, structured debt optimization, and personalized credit coaching with complete digital transparency.

## 🌟 Key Features

*   **100% Digital Process**: Seamlessly request advisory and loan assistance completely online with zero physical paperwork.
*   **No Collateral Required**: Clear guidance on unsecured loan products and flexible lines of credit with no guarantor needed.
*   **Quick Approval & Disbursal**: High-velocity underwriting advisory helping clients secure swift sanction and bank transfers.
*   **Dynamic EMI Calculator**: High-fidelity interactive tool with responsive visual indicators to calculate principal, interest, and monthly amortization schedules.
*   **Expert Board Advisory**: Direct insights from our veteran legal and financial board committed to protecting and compounding your legacy.
*   **Comprehensive Financial Services**:
    *   **Home & Mortgage Refinancing**: Repo-linked lending rate (RLLR) audits for massive lifetime interest savings.
    *   **Overdraft Against Mutual Funds**: Fast liquidity lines without triggering premature capital gains tax.
    *   **CIBIL & Credit Score Improvement**: Strategic guidance to maintain optimal credit utilization ratios and bureau health.

## 🛠️ Technology Stack

*   **Framework**: Next.js 15 (App Router)
*   **Language**: TypeScript for complete type safety
*   **Styling**: Tailwind CSS v4 utilizing custom luxury brand design variables
*   **Animations**: Framer Motion for premium, low-latency micro-interactions
*   **Icons**: Lucide React
*   **Build Tool**: Vite / PostCSS / ESBuild compilation pipelines

## 🎨 Brand Guidelines

This project implements a bespoke, editorial design system aligning with the core brand values:

*   **Primary Gold** (`#D4AF37` / `#C59B27`): Representing wealth, premium craftsmanship, and elite advisory standards.

## 🚀 Getting Started

### Prerequisites

*   Node.js v18.x or higher
*   npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shri-kalyan-punji/platform.git
   cd platform
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## 📈 Production Build

To build the application for production deployment:

```bash
npm run build
npm run start
```
